# EventHub - Events Page Module

**Developed by:** Natnael
**Project:** Event Management Website

## 📝 Overview

This module represents the complete implementation of the **Events Discovery Page**. The development focused on transforming a hand-drawn wireframe into a high-performance, modern, and user-friendly interface with a sleek and responsive design.

---

## 🚀 Completed Tasks

Following the project requirements, the following features were successfully implemented:

### ✅ Custom Wireframe-to-Code

- Transformed the initial sketch into a clean **4-column structured card layout**:
  - Event Image
  - Event Title
  - Event Description
  - Action Buttons

### ✅ Dynamic Hero Slider

- Built a responsive hero section featuring:
  - Auto-sliding images
  - Integrated video background support
  - Smooth transitions and animations

### ✅ Search Engine

- Developed a real-time search functionality using JavaScript.
- Filters events instantly as users type.

### ✅ Category Filters

- Implemented a dual-filtering system:
  - Interactive category pills
  - Dropdown selection menu
- Categories include:
  - Music
  - Technology
  - Sports
  - Art

### ✅ Dummy Data Injection

- Created an array-based data structure containing:
  - 10 unique event entries
  - High-quality image assets
  - Event-specific descriptions and metadata

### ✅ Responsive Design

- Optimized layouts for:
  - Desktop devices
  - Tablets
  - Mobile phones
- Ensures a seamless user experience across all screen sizes.

### ✅ Dark Mode Toggle

- Added a theme switcher allowing users to:
  - Toggle between Light Mode and Dark Mode
  - Improve accessibility and user preference support
